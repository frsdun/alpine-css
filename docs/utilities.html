Spacing

The spacing utility classes offer an easy way to control padding and margins of elements

{property}{sides}-{size}@{screen-size}

{property} p = padding, m = margin
{sides} t = top, b = bottom, l = left, r = right
x = horizontal, y = vertical
{size} 0 - 6
{screen-size} @s = at small breakpoint and up
@m = at medium breakpoint and up
@l = at large breakpoint and up

Examples

p-1Padding size 1
m-3Margin size 3
pr-3Padding right size 3
mb-0Margin bottom set to zero
px-2Padding horizontal size 2
py-2Padding vertical size 3,
pr-3@lPadding right size 3, only on large screens and up
mb-2@sMargin bottom size 2, only on small screens and up
ml-autoMargin left set to auto
mr-autoMargin right set to auto
mx-autoCenter horizontally. Margins left and right set to auto

Default sizes

  • 0 = None
  • 1 = 1rem / 10px
  • 2 = 2rem / 20px
  • 3 = 3rem / 30px
  • 4 = 4rem / 40px
  • 5 = 6rem / 60px
  • 6 = 8rem / 80px
