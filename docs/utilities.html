Spacing

The spacing utility classes offer an easy way to control padding and margins of elements

{property}{sides}-{size}@{screen-size?}
{property} p = padding, m = margin
{sides} t = top, b = bottom, l = left, r = right
x = horizontal, y = vertical
{size} 0 - 6
{screen-size?} empty = at all screen sizes
@s = at small breakpoint and up
@m = at medium breakpoint and up
@l = at large breakpoint and up

Examples

p-1 Padding size 1
m-3 Margin size 3
pr-3 Padding right size 3
mb-0 Margin bottom set to zero
px-2 Padding horizontal size 2
py-3 Padding vertical size 3
pr-3@l Padding right size 3, only on large screens and up
mb-2@s Margin bottom size 2, only on small screens and up
ml-auto Margin left set to auto
mx-auto Center horizontally. Margins left and right set to auto

Default sizes

  • 0 = None
  • 1 = 1rem / 10px
  • 2 = 2rem / 20px
  • 3 = 3rem / 30px
  • 4 = 4rem / 40px
  • 5 = 6rem / 60px
  • 6 = 8rem / 80px

Width

Utility classes to set the width of elements or the width of an elements children 

{child?}-w-{size}@{screen-size?}
{child?} Optionally target the children
{size} Fixed: s, m, l, xl, xxl, xxxl
Fraction: 1-1, 1-2, 1-3, 1-4, 1-5, 1-6, 2-3, 3-4, 2-5, 3-5, 4-5, 5-6
Special: auto
{screen-size?} empty = at all screen sizes
@s = at small breakpoint and up
@m = at medium breakpoint and up
@l = at large breakpoint and up

Examples

w-m Width medium
w-m@l Width medium on large screens and up
w-1-2 Width half (50%) of parent
w-1-1 100% width of parent
w-3-5 Width three fifths or parent width
child-w-1-2 Child width all 50% of parent
child-w-1-6@s Child width one 6th of parent on small screens and up
w-auto Default width

Default sizes

  • s = 15rem / 150px
  • m = 30rem / 300px
  • l = 45rem / 450px
  • xl = 64rem / 640px
  • xxl = 96rem / 960px
  • xxxl = 120rem / 1200px

Flex

Utility classes to set the display property to flex-box
All flex classes can have the @s, @m and @l screen-size selector added 

flex-{option}@{screen-size?}

Horizontal alignment

flex-start Justify content left
flex-center Justify content center
flex-end Justify content right
flex-between Justify content between
flex-around Justify content around
flex-evenly Justify content evenly

Vertical alignment

flex-top Justify content top
flex-middle Justify content middle
flex-bottom Justify content bottom
flex-stretch Justify content stretch
flex-baseline Justify content baseline

Direction

flex-inline Display inline flex-box
flex-column Display flex-box vertically

Wrap and Reverse

flex-wrap Wrap items
flex-wrap-reverse Wrap items in reverse
flex-reverse Reverse the items
flex-column-reverse Reverse the items in columns

Display

d-{option}
d-block Display block
d-inline Display inline
d-inline-block Display inline block

Position

position-{option}
position-absolute Position absolute
position-relative Position relative
position-static Position static
position-sticky Position sticky
position-inherit Position inherit

Text

text-{option}
text-center Align text center
text-left Align text left
text-right Align text right
text-upper Text to upper case
text-lower Text to lower case
text-caps Text to all caps

Visibility

hide Display set to none on all screens
hide@s Display set to none on small screens and up
hide@m Display set to none on medium screens and up
hide@l Display set to none on large screens and up
